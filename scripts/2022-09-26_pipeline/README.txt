This directory contains the pipeline of files used to examine the persistence of cord blood DNA methylation (DNAm) changes associates with prenatal NO2 exposure at age one. It also includes code to identify and examine postnatal-specific DNAm changes. 

The files are prefixed with numbers 1-11 which indicate the order they should be run in. Files are in RMD format and use here() directory calling to ensure reproducibility. Functions are called from the main "script" folder.
